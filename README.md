# Wikipedia Hitler finder

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

Wikipedia Hitler finder is script in java which found the shortest path between any wikipedia page and Adolf Hitler's page.

  - Get the shortest path
  - Avoid easy path with date and location 
  

# New Features!

  - Avoid date and location option. 

### Installation

Wikipedia Hitler finder requires [java](https://www.java.com/fr/) v11+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ cd Wikipedia-Hitler-Finder  
$ java -jar hitlerFinder.jar [link] [-ban]
```
